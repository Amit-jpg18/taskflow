# 🚀 TaskFlow

TaskFlow is a full-stack task management web application built using the MERN stack.  
It allows managers and employees to efficiently create, assign, and track tasks with secure role-based authentication.

---

## 🛠️ Tech Stack

- Frontend: React
- Backend: Node.js & Express.js
- Database: MongoDB
- Authentication: JWT
- Password Security: bcrypt

---

## ✨ Features

- 🔐 Role-based Login (Manager & Employee)
- 📋 Task Creation & Assignment
- ✅ Task Status Tracking
- 👥 Employee & Department Management
- 🔒 Secure Authentication using JWT
- 📁 File Upload Support
